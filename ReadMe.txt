Proiectul este realizat in Visual Studio 2013 deci necesita instalarea a vcredist 2013, varianta 86 si 64
De asemenea, proiectul utilizeaza fonturi speciale, ce vor fi gasite in folderul Fonts

Instructiuni:
Sarpele se misca o data la fiecare 500ms
Pe modul singleplayer mancarea valoreaza 1 punct, pe celelalte 2 puncte
Pe langa mancare, pe ecran apar si alte imagini:
 Caspuna: Dubleaza scorul si adauga 1 punct la scor
 Cafea: Dubleaza viteza sarpelui pentru un timp si adauga 1 punct la scor
 Ciuperca: Injumatateste viteza sarpelui pentru un timp
 Usturoi: Injumatateste scorul

Butoanele pentru a controla sarpele sunt WASD si 8456 (de pe numpad, pentru doubleplayer)
